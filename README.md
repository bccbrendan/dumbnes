# dumbnes
A dumb nes emulator

# depenencies

## cmake
```sh
sudo apt install cmake
```

## readline
```sh
sudo apt-get update
sudo apt install libreadline-dev
```

## SFML
```sh
https://www.sfml-dev.org/tutorials/2.5/compile-with-cmake.php
sudo apt install libsfml-dev
```


## On Debian Linux, install compiler
sudo apt install g++


## To build:
cmake .
make
